# 📊 CSE Trading Analytics Dashboard

This project is an interactive web-based dashboard developed using **Streamlit** to analyze annual trading data from the **Colombo Stock Exchange (2006–2022)**. It integrates data visualization, statistical modeling, and time series forecasting to provide meaningful insights into market and company-level behavior.

---

## 🚀 Features

- 📈 Interactive data exploration with filters (year, industry, etc.)
- 📊 Visualizations for market trends and distributions
- 📉 Panel Regression (Fixed Effects) for company-level analysis
- 🔮 Time Series Forecasting for future value prediction
- 🧠 Clear and simple interpretations for non-technical users

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas & NumPy
- Plotly
- Statsmodels
- Linearmodels
- SciPy
